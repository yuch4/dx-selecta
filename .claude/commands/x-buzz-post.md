# X バズ投稿生成

X（旧Twitter）のバズ投稿を作成する。23種類の投稿パターンから最適な5つを選び、エンゲージメントを最大化する投稿文を生成。

## 入力

話題・テーマ: $ARGUMENTS

（空欄の場合は収集データから選択）

## 参照

- x-buzz-post-generatorスキル: `.claude/skills/x-buzz-post-generator/SKILL.md`
- 投稿パターン詳細: `.claude/skills/x-buzz-post-generator/references/patterns.md`
- 出力フォーマット: `.claude/skills/x-buzz-post-generator/references/output_format.md`
- 収集データ: `X運用/収集データ/`
- 生成データ（ナレッジ）: `X運用/生成データ/`

---

## 実行手順

### 0) ナレッジの確認（精度向上）
`X運用/生成データ/` 配下のファイルを確認し、以下を分析:
- よく選ばれているパターンの傾向
- トピック×パターンの相性
- ユーザーの文体・トーンの好み

### 1) トピックの特定

**トピックが明確な場合**:
- 入力されたトピックをそのまま使用

**トピックが曖昧または空欄の場合**:
1. `X運用/収集データ/` 配下のファイル一覧を確認
2. ファイル名・タイトルからトレンド性・話題性の高いテーマを1つ選択
3. 選択したファイルを読み込み、投稿ネタを抽出
4. 選択理由をユーザーに説明

### 2) トピック分析
1. トピックの本質と拡散ポイントを分析
2. ターゲット層（共感しやすい属性）を想定
3. どの感情を刺激するか決定（共感/驚き/怒り/笑い/感動）

### 3) パターン選択
スキルの23種類のパターンから、トピックに最適な5つを選択

### 4) 投稿文生成
選択した5パターンでそれぞれ投稿文を作成

**作成ルール**:
- 文字数は300字程度（日本語基準）
- 絵文字は控えめに（0-2個程度）
- こまめに改行を効果的に使う
- 句読点は使わない
- 炎上リスクのある表現は避ける
- 特定の個人・団体を攻撃しない

### 5) 出力

スキルの出力フォーマットに従い、5種類の投稿案を出力

- 投稿時間：平日12-13時、20-22時が狙い目
- 画像・動画を添えると効果UP
- リプライにも積極的に返すと拡散しやすい
