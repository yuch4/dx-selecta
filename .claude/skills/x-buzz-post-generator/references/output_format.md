## パターン選択の指針

トピックの性質に応じて最適なパターンを選択する：

| トピックの性質 | おすすめパターン | Grok評価向け理由 |
|--------------|----------------|-----------------|
| 自分の成長・変化を伝えたい | 時系列変化、ビフォーアフター、やめたら良くなった系 | 独自体験＝唯一無二の文脈 |
| データ・事実を共有したい | 衝撃の数字＋解説、衝撃の真実、よくある勘違い | 「なぜ重要か」が明確 |
| ノウハウを教えたい | 5選リスト、質問→回答、やるな・やれ | 専門性の深さをアピール |
| 意見を述べたい | 逆張り意見、引用＋意見、気づきシェア | 独自視点がGrok評価に直結 |
| エンゲージメントを高めたい | 比較表、問いかけ、行動喚起 | 議論喚起で文脈が深まる |
| 本音・ストーリーで惹きつけたい | 告白・本音、ストーリーフック、逆張り＋体験談 | 独自性＋感情で差別化 |

## 新アルゴリズム対応チェック

生成した各投稿が以下を満たしているか確認：

| チェック項目 | 説明 |
|-------------|------|
| 文脈がある | リンクだけでなく「なぜ重要か」を説明 |
| 面白さが明確 | 「どこが面白いか」を具体的に指摘 |
| 意見がある | 情報再配布でなく独自の考察 |
| ニッチな専門性 | 万人受けより特定層に深く刺さる |
| 視覚情報 | 可能なら画像・図解を推奨 |
