## 基本パターン

### 共感・あるある型
多くの人が「わかる」と感じる視点で共感を集める

```
〇〇って結局△△だよね

分かる人には分かる
```

### 驚き・意外性型
予想を裏切る切り口、「実は」要素で興味を引く

```
〇〇の真実、意外すぎた

実は△△だったらしい

これ知ってた？
```

### 物語・体験型
個人的エピソード風、起承転結を凝縮

```
〇〇したら△△になった話

きっかけは□□で

結果的に◇◇まで変わった

人生何があるか分からない
```

### 問いかけ・議論喚起型
フォロワーの意見を引き出す、二択提示

```
正直、〇〇ってどう思う？

A：△△
B：□□

自分は◇◇派なんだけど
```

### ユーモア・ネタ型
自虐・誇張・比喩、皮肉やツッコミ

```
〇〇あるある

△△して□□になるやつ

俺だけじゃないはず
```

## 体験・変化系

### 逆張り＋体験談
一般的な意見の逆を主張し、自分の体験で裏付ける

```
〇〇は△△だと思われがちだけど

実際にやってみたら全然違った

結論：□□だった

これマジで知っておいてほしい
```

### 時系列変化
過去と現在を対比させて変化を強調

```
1年前の自分：〇〇

今の自分：△△

何が変わったかというと

□□を始めたから

これだけで人生変わる
```

### やめたら良くなった系
何かをやめたことで得られた成果を共有

```
〇〇やめたら人生変わった

・△△が改善
・□□も良くなった
・◇◇まで手に入った

なぜもっと早くやめなかったのか
```

### ビフォーアフター
変化を対比させて印象を強める

```
【〇〇する前】


【〇〇した後】


この差、エグい。
```

### ストーリーフック
物語形式で続きが気になる構成

```
〇〇年前の自分に言いたい。

「△△だけは絶対やめとけ」

なぜなら...


今の自分があるのは〇〇のおかげ。
```

## リスト・比較系

### 5選リスト
読みやすいリスト形式で情報を整理

```
【〇〇な人がやっている5つのこと】

①

②

③

④

⑤

最後が一番大事
```

### ランキング
順位をつけて紹介

```
【〇〇ランキング TOP5】

1位：
2位：
3位：
4位：
5位：

異論は認める
```

### 比較表
2つの選択肢を比較

```
〇〇派 vs △△派

【〇〇派】
・
・
・

【△△派】
・
・
・

あなたはどっち？
```

### よくある勘違い
意外な事実で興味を引く

```
〇〇について、よくある勘違い

❌


✅ 実は


これ知らない人多すぎ。
```

```

❌
❌
```

## 意見・本音系

### 告白・本音
本音を語って共感を得る

```
正直に言います。

〇〇って思ってた時期がありました。

でも実際やってみたら...


人生変わった。
```

### 引用＋意見
名言や他人の意見に自分の意見を加える

```
「〇〇」

これ、〇〇さんの言葉なんだけど

本当にそう思う。

なぜなら...


あなたはどう思いますか？
```

### 逆張り意見
一般的な意見の逆を主張

```
これ言うと怒られるかもだけど

〇〇って実は△△だと思う。

理由を説明すると...


反論あったら教えてください。
```

### 気づきシェア
最近気づいたことを共有する

```
最近気づいたんだけど

〇〇な人って△△してる

逆に□□な人は◇◇してる

この違い、めちゃくちゃ大きい
```

## 情報・行動系

### 衝撃の数字＋解説
インパクトのある数字で興味を引き、解説する

```
〇〇の人は△△%いるらしい

これってつまり...

□□ということ

みんなもっと意識した方がいい
```

### 衝撃の真実
意外性のある情報で注目を集める

```
これ、マジで衝撃だったんだけど

〇〇って実は△△らしい。

つまり...


知ってました？
```

### 質問→回答
よくある質問に回答する形式

```
「〇〇ってどうやるんですか？」

よく聞かれるので答えます。

結論：


理由は3つあって

①
②
③

参考になったらいいね
```

### 行動喚起
読者に具体的な行動を促す

```
〇〇したい人、今すぐこれやって。

①

②

③

やった人は「やった」とリプください。
応援します
```
