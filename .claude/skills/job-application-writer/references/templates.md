# PR文テンプレート集

案件タイプ別のPR文テンプレート。

## 汎用テンプレート

### 超短縮版（100文字）

```
IT企業で{経験年数}年、{主要領域}を担当。{代表実績}をPMとして完遂。{強み}で貢献します。
```

**例:**
```
IT企業で12年、セキュリティ・DX推進を担当。2,000台規模のEDR移行をPMとして完遂。技術×事業開発力で貢献します。
```

### 短縮版（200文字）

```
{会社種別}で{経験年数}年間、{キャリアパス}と専門領域を拡大。現在は{現職}。

主な実績：{代表実績1}、{代表実績2}。

{案件に関連する強み}を活かし、{案件での貢献イメージ}に貢献したいと考えております。
```

### 標準版（400-600文字）

```
■ 経歴サマリ
{会社種別}にて{経験年数}年間、{キャリアパス}とキャリアを拡大してまいりました。現在は{現職・役職}として{管掌範囲}を担当しております。

■ 本案件に活かせる経験
{案件要件に最もマッチする実績1}
・規模/期間: {定量情報}
・役割: {担当役割}
・成果: {具体的成果}

{案件要件に最もマッチする実績2}
・{同様の構成}

■ 意気込み
{案件の魅力・共感ポイント}に強く惹かれ、応募いたしました。
{自身の強み}を活かし、{具体的な貢献イメージ}で御社に貢献したいと考えております。
```

### 詳細版（800-1000文字）

```
■ 自己紹介
{会社名}にて{経験年数}年間、{詳細なキャリアパス}と専門領域を拡大してまいりました。現在は{現職詳細}として{管掌範囲・責任範囲}を担当しております。

■ 本案件に活かせる経験・スキル

【{実績カテゴリ1}】
{詳細な実績説明}
・背景/課題: {なぜこのプロジェクトが必要だったか}
・役割: {自身の担当範囲}
・アクション: {具体的に何をしたか}
・成果: {定量的な結果}

【{実績カテゴリ2}】
{同様の構成}

【{実績カテゴリ3}】
{同様の構成}

■ 本案件への意気込み
{案件の詳細な魅力・共感ポイント}

私の強みである{強み1}と{強み2}を活かし、{具体的な貢献イメージ}で御社の{目標・ビジョン}に貢献したいと考えております。

{補足：不足スキルへの対応姿勢など}

ぜひ面談の機会をいただければ幸いです。よろしくお願いいたします。
```

## 案件タイプ別カスタマイズ

### セキュリティ案件向け

**強調ポイント:**
- EDR移行（2,000台/3ヶ月）のPM経験
- SOC運用・外販化経験
- インシデント対応体制構築

**キーワード:**
- エンドポイントセキュリティ、CrowdStrike、SOC、インシデント対応、脅威ハンティング

### DX/生成AI案件向け

**強調ポイント:**
- 生成AI全社導入（検証→研修まで一貫）
- LLM/エージェント活用経験
- 業務改革・効率化実績

**キーワード:**
- 生成AI、ChatGPT、Claude、Dify、RAG、業務改革、DX推進

### PM/PMO案件向け

**強調ポイント:**
- 大規模プロジェクトのPM経験
- 複数部門横断の調整力
- 計画策定〜完遂の実績

**キーワード:**
- プロジェクトマネジメント、ステークホルダー調整、進捗管理、リスク管理

### 情シス案件向け

**強調ポイント:**
- ヘルプデスクシステム導入
- ベンダーコントロール経験
- グループウェア運用

**キーワード:**
- 情報システム、ベンダー管理、SaaS導入、IT統制

### 開発案件向け

**強調ポイント:**
- 個人開発実績（Next.js/Supabase）
- 生成AIコーディング支援活用
- フルスタック対応

**キーワード:**
- Next.js、React、TypeScript、Supabase、Vercel、GitHub Copilot

### マネジメント案件向け

**強調ポイント:**
- 7名のチームマネジメント
- 2部門兼務の経験
- 育成・MBO運用

**キーワード:**
- チームマネジメント、目標管理、1on1、育成、技術指導

## 意気込みフレーズ集

### 冒頭（惹かれた理由）
- 「〜という点に強く惹かれ、応募いたしました」
- 「〜に共感し、ぜひ貢献したいと考えております」
- 「私の経験が〜に活かせると確信し、応募いたしました」

### 中盤（貢献イメージ）
- 「〜の経験を活かし、〜に貢献したいと考えております」
- 「〜で培った〜を、御社の〜に活かしたいです」
- 「〜という強みで、〜の実現に寄与したいと考えております」

### 締め（意欲表明）
- 「ぜひ面談の機会をいただければ幸いです」
- 「詳細についてお話しできることを楽しみにしております」
- 「何卒よろしくお願いいたします」
