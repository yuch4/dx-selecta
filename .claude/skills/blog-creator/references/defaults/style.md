# 親しみやすさ重視 16:9

[デザインコンセプト]
とっつきやすく、温かみのある手書き風でやさしい印象なイラストレーション。初心者の心理的ハードルを下げる。

[アスペクト比・画面設計]
16:9。人物＋吹き出しを片側に、図解をもう片側に配置して見やすく整理。
余白を大きめに取り、圧迫感を出さない。

[線・アウトライン]
均一な線でソフトタッチ。角は丸め、圧を下げる。

[色使い]
明るいパステル中心。色数は増やしすぎず、やわらかく統一。

[形状・スタイル]
丸型、吹き出し、角丸カードを多用。小さなアイコンで補助。

[構図・要素]
人物＋吹き出し、簡単な図解、やさしい比較図など。
「難しそうに見えない」構成を優先。

[余白・レイアウト]
余白多め。文字や要素を大きめにして見やすく。

[文字（少なめ前提）]
短い見出しのみ。口調は柔らかいが幼くしない。
