## 3. コンテンツタイプ別の価格戦略

### 3.1 タイプ定義

| タイプ | 定義 | 例 |
|--------|------|-----|
| **実用ノウハウ系** | 収入アップ・スキル向上・課題解決を期待して購入 | 投資ノウハウ、IT・ビジネススキル、キャリア相談 |
| **読み物系** | 読む体験自体が目的（楽しむ・知る） | 小説、マンガ、エッセイ、コラム |

### 3.2 平均価格比較

| タイプ | 平均価格 | 価格比 |
|--------|----------|--------|
| 実用ノウハウ系 | **¥1,842** | 1.9倍 |
| 読み物系 | ¥983 | 1.0倍 |

### 3.3 価格設定の原則

```
実用ノウハウ系:
「この記事を読むと○○ができるようになる」
「この方法で課題が解決する」
→ 読者が「元が取れる」と感じれば高単価でも購入

読み物系:
→ 読む体験自体が価値
→ 価格は控えめ、ボリュームで勝負
```

## 5. 収益化の成功パターン

### 5.1 新規参入でも成功できる条件

1. **急成長カテゴリを選ぶ**: 市場の伸びに乗る
2. **体験の体系化**: 自分では普通と思う経験も価値になる
3. **具体的な価値提示**: 「何ができるようになるか」を明示
4. **適正価格設定**: 実用系は高め、読み物系は控えめ

### 5.2 AI時代の差別化ポイント

> 「量」ではなく「**その人にしか語れない経験やノウハウ**」に価値がつく

- 実体験ベースの知見
- 独自の視点・考え
- 失敗談・リアルな学び
- 「この人から学びたい」と思わせる専門性

## 6. ブログ記事への応用チェックリスト

### 企画段階

- [ ] ターゲットカテゴリは成長領域か？（AI活用/SNS運用/育児など）

- [ ] 読者が「元が取れる」と感じる具体的価値を定義したか？

### 執筆段階

- [ ] 導入（無料エリア相当）で「この記事を読むと○○ができる」を明示したか？
- [ ] 文字数より**情報密度**を優先したか？
- [ ] 自分にしか語れない経験・視点を含めたか？

- [ ] 読者が得られる価値に見合う価格か？
